# 🌤️ Clima en Chile - Dashboard de Pronóstico

Una aplicación web interactiva que permite visualizar el estado del clima actual y el pronóstico semanal de las principales ciudades de Chile. Este proyecto utiliza JavaScript para la manipulación dinámica del DOM y Bootstrap 5 para un diseño responsivo y moderno.

## 🚀 Características

* **Grilla Dinámica:** Visualización de más de 10 ciudades con su clima actual (temperatura, humedad, condición).
* **Vista de Detalle:** Al hacer clic en una ciudad, se despliega información extendida y el pronóstico para los próximos 7 días.
* **Interfaz Responsiva:** Adaptable a dispositivos móviles, tablets y computadoras gracias a Bootstrap.
* **Iconografía Dinámica:** Uso de Font Awesome para representar las condiciones climáticas de forma visual.
* **Navegación Fluida:** Cambio entre la vista principal y de detalle sin recargar la página (SPA logic).

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **CSS:** Estilos personalizados y animaciones hover en las tarjetas.
* **JavaScript:** Lógica de renderizado, manejo de eventos y gestión de arreglos de objetos.
* **Bootstrap:** Framework para el diseño y componentes responsivos.
* **Font Awesome:** Biblioteca de iconos.
